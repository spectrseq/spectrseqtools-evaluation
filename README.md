# Evaluation of mass spec based RNA sequencing

To run this workflow, please install Conda and Snakemake.
In addition, clone the github repository https://github.com/koesterlab/lionelmssq to some place (let's call that path `<lionelmssq_repo>` below).
Install lionelmssq into the same environment as your Snakemake via

```bash
pip install -e <lionelmssq_repo>
```
